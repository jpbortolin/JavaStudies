# ☕ Java Fundamentos — Capítulo 1: Orientação a Objetos

Este capítulo apresenta os conceitos iniciais da linguagem Java e os fundamentos da Programação Orientada a Objetos (POO).

## Introdução

A tecnologia está presente em praticamente tudo, e os sistemas precisam de linguagens de programação para serem desenvolvidos. Entre as principais linguagens do mercado, o Java se destaca por sua portabilidade, simplicidade e uso da Orientação a Objetos.

Além disso, aprender Java ajuda a entender conceitos que também são utilizados em outras linguagens, como C#, PHP e Python.

## Introdução ao Java

O Java foi criado em 1995 pela Sun Microsystems, liderada por James Gosling.

### Principais características:
- Simples
- Robusto
- Seguro
- Alto desempenho
- Portável
- Orientado a Objetos

Hoje o Java é utilizado em sistemas web, aplicações corporativas, dispositivos móveis e diversos outros ambientes.

## Portabilidade

Uma das maiores vantagens do Java é a portabilidade:

> “Escreva uma vez e execute em qualquer lugar.”

O código Java é compilado em **bytecode**, que é executado pela **JVM (Java Virtual Machine)**.  
Isso permite que o mesmo programa funcione em diferentes sistemas operacionais sem precisar recompilar.


## Orientação a Objetos

A POO organiza os sistemas em **classes** e **objetos**.

- **Classe:** molde/modelo
- **Objeto:** instância criada a partir da classe

### Exemplo:
- Classe: `Carro`
- Objetos: Uno, Civic, Ferrari

Cada objeto possui:
- **Atributos:** características (cor, modelo, potência)
- **Métodos:** ações/comportamentos (acelerar, frear, ligar)

## Princípios da Orientação a Objetos

### Abstração
Representa apenas as informações importantes para o sistema.

### Encapsulamento
Protege os dados do objeto, permitindo acesso controlado.

### Herança
Permite que uma classe herde características de outra.

Exemplo:
- `Pessoa` → `Aluno`

### Modularização
Divide o sistema em partes menores e reutilizáveis, facilitando manutenção e organização.
